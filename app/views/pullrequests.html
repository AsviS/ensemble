Contribute code!

If you want to contribute to the code directly (to implement one of the suggested features or an idea of your own), you can do so by making a pull requests to the Github repository of the project (no need to open an issue first).

If you don't have experience with pull requests, have a look at Thinkful's great tutorial on the subject.

In the future, this tab will display more information about pull requests using the pull request API.
